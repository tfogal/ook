NAME

ookbricks - reports the total number of bricks in a dataset  

SYNOPSIS

#include <ook.h>

size_t ookbricks(const struct ookfile* of);

 

DESCRIPTION

ookbricks() reports the total number of bricks in a dataset. This is a simple calculation based on the total number of voxels and the brick size, but this is provided for convenience nonetheless.

 

RETURN VALUE

ookbricks() returns the total number of bricks in a dataset.

 

ERRORS

EINVAL
The given of pointer does not point to a valid struct ookfile
